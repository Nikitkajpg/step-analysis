# STEP Analysis

A **Kotlin Multiplatform** STEP file parser with a desktop UI built using **Compose Multiplatform** and **Material 3**.

The project is designed to parse `.step` and `.stp` files, inspect entity structure, and generate G-code.

## Features

- Parse STEP files (`.step`, `.stp`).
- Extract entities, identifiers, and references.
- Extract a compactor from a file.
- Generate the G-code for the coordinate manipulator.

## Tech Stack

- Kotlin Multiplatform
- Compose Multiplatform
- Material 3
